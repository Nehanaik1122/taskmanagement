# 📝 Task Management App

A full-stack, responsive task management application built with **Next.js 15**, **TypeScript**, and **Firebase**.

## 🚀 Live Demo
**View the project live here:** [https://taskmanagement-uwsy.vercel.app](https://taskmanagement-uwsy.vercel.app)

---

## ✨ Features
* **Secure Authentication:** User login and signup powered by Firebase Auth.
* **Real-time Task Management:** Create and view tasks stored in the cloud.
* **Advanced Filtering:** Sort through tasks by status: *To Do, In Progress, Completed,* and *Pending*.
* **Modern UI:** Built with Tailwind CSS and Shadcn UI components for a clean, professional look.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.

## 🛠️ Tech Stack
* **Framework:** [Next.js](https://nextjs.org/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Database/Auth:** [Firebase](https://firebase.google.com/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Deployment:** [Vercel](https://vercel.com/)
